# Task Manager (Java CLI App)

A simple command-line Task Manager built using Java.  
It allows users to add, view, and delete tasks using a menu-driven interface.

---

## Features
- Add new tasks
- View all saved tasks
- Delete tasks by number
- Continuous menu loop until exit

---

## Technologies Used
- Java
- ArrayList
- Scanner
- Loops and conditionals

---

## How it works
The application runs in the console and uses a menu system to interact with the user.  
Tasks are stored in memory using an ArrayList.
