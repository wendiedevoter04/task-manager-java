Task Manager (Java CLI)

This is a simple command-line task management application built in Java. It is designed to simulate how users can create, view and manage tasks through an interactive terminal-based interface.

The project focuses on reinforcing Java fundamentals, especially user input handling, loops and dynamic data structures.

What this project does

The application allows a user to manage a list of tasks directly from the terminal. Users can add new tasks, view existing ones and delete tasks based on their position in the list.

It runs continuously in a menu-driven loop until the user chooses to exit.

Key features

Add new tasks to the system
View all saved tasks
Delete tasks by selecting their number
Continuous interactive menu loop

Tech stack

Java
ArrayList
Scanner (user input handling)
Loops and conditional statements

How it works

The program runs in a continuous loop, displaying a menu of options to the user. Based on the selected option, the system performs actions such as adding, displaying or removing tasks.

All tasks are stored in an ArrayList, which allows dynamic resizing as items are added or removed.

User input is captured using the Scanner class and processed through conditional logic.

What I learned

This project helped me understand how to build interactive console applications in Java. I learned how to manage user input, structure program flow using loops and handle dynamic data using ArrayLists.

It also improved my understanding of building simple state-based systems where user actions directly modify stored data.

Project summary

A Java-based command-line task manager that allows users to add, view and delete tasks. The project demonstrates foundational programming concepts including user input handling, loops and dynamic data structures using ArrayList.
