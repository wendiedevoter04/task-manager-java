# Task Manager (Java CLI App)

This is a simple command-line task manager I built to practice Java fundamentals and understand how user input and data structures work together in an interactive program.

The goal was to create a small but functional system that runs entirely in the terminal.

## Features

- Add new tasks
- View all saved tasks
- Delete tasks by number
- Runs in a continuous menu loop until exit

## Technologies Used

- Java
- ArrayList
- Scanner
- Basic control structures (loops and conditionals)

## How It Works

The application runs in the console and presents a menu to the user.  
Tasks are stored in an ArrayList, and users can interact with them by selecting options from the menu.

## What I Learned

While building this project, I learned how to:
- Handle user input in Java using Scanner
- Manage dynamic data using ArrayLists
- Build a simple interactive program with a menu system
